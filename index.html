    SabdroidEx

    Project maintained by hellhand Hosted on GitHub Pages — Theme by mattgraham

    Description

    SABDroidEx is a frontend for sabnzbd+, Sickbeard and Couchpotato on Android. It is used to monitor your servers and to interact with them in the most complete and intuitive way.

    SabdroidEx uses this library : http://viewpagerindicator.com/ Market URL: https://market.android.com/details?id=com.sabdroidex

    Beta participation

    To participate to the beta and have the most up-to-date version of SabdroidEx, join this community https://plus.google.com/u/0/communities/111434829408878579692 and follow the link in the first post available.

    General Functions

    The servers can be contacted behind an Apache web server, separate authentication and access through SSL available, following Android 4.0+ UI Look & Feel.

    Sabnzbd+ Functions

    See the list of downloads, see the history, the downloads can be separately paused or cancelled, the refresh rate can be changed, items can be removed from the History, downloads can be added with a direct link to an nzb, downloads can be added by opening an nzb file, Sabnzbd+ server settings can be accessed and changed.

    SickBeard Functions

    See the list of Shows, see the seasons for each Show, see the status for each Episode of a Season, change the status for each Episode.

    CouchPotato Functions

    See the list of Movies. and much more to come ...
